## Github Issues
Работа с обращениями нужного репозитория.
- React
- Mobx
- CSS Modules
---
Установка зависимостей:
```bash
npm install #yarn install
```
Чтобы запустить проект локально:
```bash
yarn start # npm start
# localhost:3000
```

Чтобы собрать проект:
```bash
yarn build # npm run build
```
